# weaviate-sandbox-ts

Loosely following https://weaviate.io/developers/academy/

## Installation

### Install nvm

### Install direnv

### Run the following

```sh
git clone git@github.com:foresight-host/weaviate-sandbox-ts.git
cd weaviate-sandbox-ts
npm i -g pnpm
pnpm i
cp .env.sample .env
# Populate environment values in .env
direnv allow
```

## Run code exercises

```sh
pnpm 1.7-hands-on
```
